# mdBook Mermaid Pan/Zoom Preprocessor

A preprocessor for mdBook that adds pan/zoom functionality to mermaid diagrams.

## Features

- Interactive pan and zoom for mermaid diagrams
- Fit-to-view and fullscreen modes
- Responsive design with dark theme support
- Easy asset management with `generate` command

## Installation

1. Install the preprocessor:
   ```bash
   cargo install --path .

2. Install into your mdBook project:

bash

cd your-book-directory
mdbook-mermaid-panzoom install
